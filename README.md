# Wave

A **Geometry Dash Wave** style game for **NumWorks calculators**.


  <img src="doc/game.gif" width="300">


The game includes **3 difficulty levels**:
- Easy
- Medium
- Hard

Your **best scores are saved** for each difficulty.

Try the old python version : **https://my.numworks.com/python/artlislisart/geometry_dash_wave**

## Controls

**`OK`**        -> Jump

**`RETURN`**  -> Pause the Game 

**`CLEAR`**   -> Go to Start Menu

## Installation

1. Download the `wave.nwa` file from the **Releases** section.
2. Connect your **NumWorks calculator** to your computer.
3. Go to **https://my.numworks.com/apps**.
4. Drag and drop the `wave.nwa` file onto the page.
5. Install the app on your calculator. (⚠️ You calculator may crash the first time you install it, make sure you save all data)
